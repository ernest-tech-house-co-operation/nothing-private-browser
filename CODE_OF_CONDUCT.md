# Code of Conduct

## The short version

Be respectful. Don't be a jerk. That's it.

## The longer version

- Criticism of code is fine. Personal attacks are not.
- If you disagree with a decision, say so clearly and move on.
- We reserve the right to close issues or ban users who make the space unpleasant.
